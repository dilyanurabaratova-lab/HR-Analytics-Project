# HR-Analytics-Project
HR Analytics project using SQL and Power BI to analyze employee attrition and workforce metrics.
📊 HR Analytics Dashboard
Project Overview

This project analyzes employee attrition using SQL Server and Power BI. The objective is to identify factors influencing employee turnover and provide insights that support HR decision-making.

Tools Used
SQL Server
Power BI
DAX
Power Query
Excel
Dataset

IBM HR Analytics Employee Attrition dataset

1470 employees

Business Questions

This project answers the following questions:

What is the overall attrition rate?
Which department has the highest employee turnover?
Does education affect attrition?
Does marital status influence attrition?
How does salary differ between employees who stay and leave?
How long do employees stay before leaving?
Does job satisfaction affect attrition?
Does environment satisfaction affect attrition?
Does work-life balance affect attrition?
Which employees belong to the highest risk group?
Does commuting distance influence attrition?
Does changing companies frequently increase attrition?

Key Insights
Overall employee attrition is 16.1%.
Sales has the highest attrition rate.
Employees commuting more than 16 km have higher turnover.
Lower job satisfaction is associated with higher attrition.
Employees with lower salaries are more likely to leave.
High-risk employees can be identified using satisfaction, work-life balance, and tenure.
